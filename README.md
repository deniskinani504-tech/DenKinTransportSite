# DenKin Transport Services Website

This repository contains the website for **DenKin Transport Services**, a premium executive transport and tour service based in Kampala, Uganda.

## Overview
DenKin Transport Services offers reliable, elegant, and professional transport solutions, including airport transfers, corporate travel, and tours. The website is designed to showcase services, provide easy contact via WhatsApp and phone, and allow customers to book or inquire quickly.

## Features
- Fully responsive website, mobile-friendly
- Direct contact via WhatsApp and phone links
- QR code ready for business cards and marketing
- Live site hosted on Netlify

## Live Site
[Visit DenKin Transport Services](https://denkin-transport-services.netlify.app)

## Deployment
This site is deployed on Netlify. To update or redeploy:
1. Clone the repository
2. Make edits locally
3. Push changes to GitHub
4. Netlify will automatically redeploy the site
